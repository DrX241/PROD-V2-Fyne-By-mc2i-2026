# Documents Directory

This directory is used to store dynamically generated documents that serve as attachments in the simulated email conversations.

Documents are generated by the document generator service and stored here temporarily. Each document is associated with a specific scenario and contains information relevant to the cybersecurity training exercise.

The document generator creates files based on templates and customizes them according to the scenario context. Documents might include:

- Log files
- Security reports
- Network diagrams
- Vulnerability assessments
- Policy documents
- Incident response forms

Each document follows a naming convention: `[scenario_id]_[document_type]_[timestamp].[extension]`

Documents are accessible to users through the simulated email interface, where they appear as clickable attachments.
