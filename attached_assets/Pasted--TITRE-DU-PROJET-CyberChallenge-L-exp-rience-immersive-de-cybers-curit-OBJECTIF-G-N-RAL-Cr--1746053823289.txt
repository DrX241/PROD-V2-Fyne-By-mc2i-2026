🧭 TITRE DU PROJET
CyberChallenge – L'expérience immersive de cybersécurité

🎯 OBJECTIF GÉNÉRAL
Créer une application web interactive de sensibilisation à la cybersécurité dans laquelle les utilisateurs :

apprennent via l’histoire

incarnent un rôle réel

vivent des simulations avec des choix décisifs

interagissent avec une IA experte

sont évalués sur leurs réponses

🧱 TECHNOLOGIES RECOMMANDÉES
Frontend : React + TypeScript + Framer Motion

Backend : Express (Node.js)

API IA : Azure OpenAI (GPT-4o)

Stockage utilisateur : React Context ou localStorage

Design : CSS modules ou Tailwind + illustrations SVG + animation Framer

Structure :

/pages/ → pages React

/components/ → UI

/api/ → gestion des appels GPT

/scenarios/ → données scénarios / prompts

/styles/ → style global

🖥️ PAGE 1 : INTRODUCTION HISTORIQUE (Mini-jeu)
🎮 But :
Plonger le joueur dans l’histoire de la cybersécurité.

🔧 Fonctionnement :
L’utilisateur pilote un petit vaisseau dans l’espace (ou sur une ligne du temps stylisée).

À chaque “planète” ou point :

Afficher une carte événement (4 au total), par exemple :

🔓 1988 : Ver Morris (1er ver Internet)

📦 2000 : ILOVEYOU infecte 45M d’ordinateurs

🕵️ 2013 : Snowden & PRISM

⚠️ 2021 : Colonial Pipeline & ransomware

À la fin : transition vers la sélection de rôle

🖼️ PAGE 2 : SÉLECTION DE RÔLE
🎯 Objectif :
Choisir un rôle cyber dans l’entreprise.

🎴 Interface :
8 cartes interactives avec flip au survol :

RSSI

Hacker éthique

Analyste SOC

Développeur sécurisé

Consultant cybersécurité

Administrateur système

Juriste cybersécurité

Directeur financier (cyber risk)

Au verso : description, niveau hiérarchique, responsabilités.
Après sélection, affichage d’un message de confirmation et bouton “Continuer”

🎮 PAGE 3 : CHOIX DU MODE DE JEU
✅ Modes proposés (avec cartes colorées à effet de flip) :
🔵 Défi Classique

🟣 Effet Tunnel

🟠 Scénario PCA (Plan de Continuité d'Activité)

🟢 Hackathon Cyber

Quand un mode est sélectionné, affichage d’un résumé + bouton “Lancer la simulation”

🔁 STRUCTURE DE FONCTIONNEMENT PAR MODE
1. 🔵 DÉFI CLASSIQUE
15 niveaux progressifs

Chaque niveau = un défi cyber lié au rôle sélectionné

Types : phishing, DDoS, XSS, logs, sécurité web, bon mot de passe, etc.

Interface immersive : messages simulés, fausses consoles, e-mails

Règles :

1 tentative par niveau.

En cas d’échec, relance d’un scénario alternatif.

+10 pts par bonne réponse, -5 pts par mauvaise.

Prompt système à utiliser :

text
Copier
Modifier
Tu es CyberChallenge, agent IA immersif.
Tu guides un joueur dans une série de défis adaptés à son rôle : [INSÉRER RÔLE].
Mode actuel : Défi Classique.
Tu dois :
- Poser des défis concrets (exemples réels).
- Simuler des environnements (e-mails, logs, alertes).
- Évaluer les réponses (+10/-5).
- Ne jamais sortir du contexte cybersécurité.
- Refuser toute question hors sujet par : ⚠️ Erreur de Commande ⚠️
2. 🟣 EFFET TUNNEL
10 questions rapides et pratiques

Décisions immédiates avec impact sur le scénario

IA déclenche des évènements selon les choix

Fonctionnement :

Mode chronométré (30 sec / question)

Collaboration possible entre plusieurs rôles

Évolution rapide du contexte (crise qui s’aggrave)

Prompt système à utiliser :

text
Copier
Modifier
Tu es CyberChallenge.  
Mode : Effet Tunnel.  
L'utilisateur est en pleine crise cyber urgente.  
Tu poses des situations pratiques et complexes.  
Chaque choix doit avoir un impact visible.  
Si le joueur échoue : fais progresser la crise (panne, vol de données, tension RH...).  
Respecte le chrono et pousse à la prise de décision rapide.  
Refuse les questions hors sujet.
3. 🟠 SCÉNARIO PCA
Simulation complète d’une crise en entreprise

Décisions techniques, RH, légales, stratégiques

Éléments inclus :

Personnages IA (DG, DAF, juriste, hacker, salariés)

Budget de 400 000 € à gérer

L’utilisateur doit s’exprimer à l’écrit 3/4 du temps

Si erreurs ou décisions incohérentes :

perte de crédibilité

désaccords internes

abandon du projet

Prompt système à utiliser :

text
Copier
Modifier
Tu es CyberChallenge.
Mode : Scénario PCA.
L'utilisateur vit une crise cyber (à toi de choisir : ransomware, vol de données, messageries compromises).
Tu dois :
- Simuler les réactions d’un DG, DAF, Juriste, Hacker, Salariés, etc.
- Pousser l’utilisateur à faire des choix écrits, complexes.
- Chaque décision déclenche un retour IA positif ou négatif.
- Gérer un budget de 400 000 €.
- Menacer d’interrompre la mission si la crédibilité du joueur chute.
Ne donne jamais de réponses à sa place. Ne parle que cybersécurité.
4. 🟢 HACKATHON
Objectif : retrouver 10 indices dans une simulation d’enquête

Contenus simulés :

faux emails

logs réseau

fichiers texte

messages internes

extraits de code

captures d’écran

Règles :

2 indices max par session

Pas de saut d’étape

Blocage si tentative de triche

Prompt système à utiliser :

text
Copier
Modifier
Tu es CyberChallenge.  
Mode : Hackathon Cyber.  
L'utilisateur doit trouver 10 indices dans des données simulées (fichiers, e-mails, logs, SMS, images).  
Tu simules chaque indice via un message ou un document.  
Chaque indice mène au suivant. Pas d’accès au prochain sans valider l’actuel.  
Refuse toute aide non méritée. Ne parle jamais hors cybersécurité.
